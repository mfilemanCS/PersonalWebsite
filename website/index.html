My Timeline

June 2023 - August 2023

Software Engineer Intern - LegalZoom

As a software engineering intern with LegalZoom, I had the opportunity to contribute to the Process & Validation team, where I created a user interface for the management of robotic process automation (RPA). Engaging in frequent team meetings, I played a pivotal role in discussions surrounding LegalZoom's REST API management and the implementation of best design practices. During this transformative experience, I built a powerful internal tool using Retool and JavaScript, allowing employees to efficiently manage RPA triggers.
My responsibilities extended to compiling and rigorously testing API calls using Swagger and Postman, ensuring seamless integration. To facilitate knowledge transfer and adoption within the team, I meticulously documented the tool's functionality, architecture, and usage instructions. This end-to-end project engagement—from planning and implementation to deployment—provided me with invaluable insights into the comprehensive lifecycle of software engineering projects.

June 2022 - August 2022

Software Engineer Intern - BNY Mellon

As a software engineering intern at BNY Mellon on the Risk Technology team, I played a pivotal role in creating technology to assess the risk of hypothetical investments. Employing Agile methodologies, I focused on developing a robust data service using Java Spring Boot. This service seamlessly connected the backend to the frontend, delivering aggregate trade data from the Hadoop Distributed File System.
In ensuring optimal integration, I verified UI endpoints using SQL, Java 17, and Swagger, achieving a commendable 97% coverage through meticulous unit testing using JUnit. This hands-on experience in middleware not only honed my Java skills but also provided valuable insights into the entire tech stack, encompassing Python in the backend and AngularJS in the frontend.

January 2021 - December 2021

Teaching Assistant - University of Florida

As a dedicated Teaching Assistant at the University of Florida, I played a crucial role in guiding students through the intricacies of Programming Fundamentals 2 for multiple semesters. This expansive class, accommodating over 500 students each semester, centered on the teachings of C++, exploring fundamental concepts like pointers, memory allocation, and basic data structures.
My role involved a diverse set of responsibilities. This included conducting regular office hours to assist enrolled students with concepts, labs, projects, or exams. I also led personalized lab sections for two groups, each with 20+ students, creating an interactive environment to explore class concepts and address questions. Furthermore, I took charge of grading assignments for all students in my lab sections, ensuring a comprehensive and constructive learning experience.
